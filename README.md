## 🧹 À propos

MenagePro est une plateforme de mise en relation entre les clients et les femmes de ménage professionnelles au Cameroun.

## ✨ Fonctionnalités

- ✅ Inscription facile pour les femmes de ménage
- ✅ Recherche géolocalisée avec Google Maps
- ✅ Paiement mobile (Orange Money & MTN Mobile Money)
- ✅ Administration en temps réel
- ✅ Notifications SMS (Twilio)
- ✅ Upload de photos
- ✅ Interface responsive

## 🚀 Technologies utilisées

- **Frontend** : HTML5, CSS3, JavaScript (ES6+)
- **Backend** : Firebase (Auth, Firestore, Storage)
- **Paiement** : Orange Money API, MTN Mobile Money API
- **SMS** : Twilio
- **Cartes** : Google Maps API
- **Hébergement** : Netlify / Vercel
